# 🥠 Fortune Cookie (C++)

A simple C++ program that simulates opening a fortune cookie in the terminal.

Each time the program runs, it prints a **random fortune message** along with **six lucky numbers**.

## 🎯 Features

* Random fortune messages
* Lucky numbers generator
* Uses `rand()` and `srand()` for randomness
* Beginner-friendly C++ project

## 🧠 How It Works

1. The program seeds the random number generator using the current time.
2. A random number between **1–6** is generated.
3. Based on that number, a **fortune message** is selected.
4. The program then prints **six random lucky numbers** between **0–49**.

## 🛠 Built With

* C++
* Standard Library (`iostream`, `cstdlib`)
